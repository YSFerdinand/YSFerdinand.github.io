植基於直方圖位移和Paillier密碼系統的可分離及可逆式之資訊隱藏方法

                                                      Abstract

Nowadays there is more and more attention on performing reversible data hiding in encrypted image, not only the original image can be perfectly recovered after extracting the embedding secret information but also the privacy of content is protected. This thesis proposes a separable and reversible data hiding in encrypted images scheme based on the histogram shifting and Paillier cryptosystem. Proposed scheme can be divided into three parts, including Image owner, Data hider and Receiver. First, Image owner takes Receiver’s public key to encrypt cover image using Paillier cryptosystem. According to the differencing between two adjacent pixels, Data hider embeds secret data into differencing applying histogram shifting and hiding key. Finally, Receiver can simultaneously extract the secret information and recover the original image if he/she has hiding key and private key. In the experimental results, it is completely shown that the proposed scheme has better performance of image quality at the same number of embedded bits than others.
Keywords: Encrypted image, Reversible data hiding, Histogram shifting, Paillier cryptosystem

Reversible data hiding in encrypted images (RDHEI) technology

    RDHEI technology combines features of Cryptography and Steganography instead of utilizing one of them unilaterally. In RDHEI technology, the cover image is encrypted firstly to produce the encrypted image. Secret information is embedded into the encrypted image in cipher domain after the encrypted image is created. Note that the content of cover image is not necessary to be known when performing data hiding. Finally, the cover image can be recovered without any error after image decryption and secret information extraction.

Paillier cryptosystem

Paillier cryptosystem is an asymmetric algorithm for public key cryptosystem. This scheme’s operation can be divided into three parts, key generation, encryption and decryption. 

The flowchart of Paillier cryptosystem is shown below.

Paillier cryptosystem is used in RDHEI to perform encryption and decryption because Paillier cryptosystem is not only a public key cryptosystem but also a homomorphic cryptosystem. In other words, Paillier cryptosystem has two homomorphic properties, additive homomorphic property and multiplicative homomorphic property, which can be utilized for data hiding in cipher domain.

Additive homomorphic property

The addition of two plaintexts is obtained after decrypting the product of two correspond ciphertexts.

D(E(m_1 )∙E(m_2 )) = D((g^(m_1 )∙r_1^n "mod" n^2)∙(g^(m_2 )∙r_2^n "mod" n^2))

                                     = D(g^(m_1+m_2 )∙(r_1∙r_2 )^n "mod" n^2) (2.3)

                                     = m_1+m_2 "mod" n

Multiplicative homomorphic property

The product of two plaintexts is gained after decrypting one corresponding ciphertext to the power of the other.

D(E(m_1 )^(m_2 ) ) = D((g^(m_1 )∙r^n "mod" n^2 )^(m_2 ) )

                                    = D(g^(m_1 m_2 )∙(r^(m_2 ) )^n "mod" n^2 ) (2.4)

                                    = m_1 m_2 "mod" n

Proposed Method

This thesis proposes a separable and reversible data hiding method in encrypted images with Paillier cryptosystem to improve the disadvantages mentioned in previous methods, including image quality of directly decrypted image is ignored and cover image may not be recovered perfectly.
The flowchart of proposed method is shown below
Image encryption
  • Partition
  • Encryption
Data Embedding
  • Divide encrypted image into blocks
  • Create the mapping table
  • Calculate the difference between two pixels
  • Embed the secret information
Data Extraction
  • Divide encrypted image with secret information into blocks
  • Create the mapping table
  • Calculate the difference between pixels with secret information
  • Data extraction
  • Image recovery
